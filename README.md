# Calculator (C + HTML + CSS)

## Description
This is a simple and functional Calculator application developed using **C** for backend computation and **HTML/CSS** for the frontend interface.  
It allows users to perform basic arithmetic operations like **addition, subtraction, multiplication, and division** through a clean and user-friendly interface.

---

## Features
- Performs **basic arithmetic operations**: `+`, `-`, `*`, `/`  
- Clean and responsive **HTML/CSS interface**  
- Simple and easy-to-use design  
- Error handling for invalid inputs (like division by zero)

---

## How to Run
### For C Program:
1. Open your C file (`calculator.c`) in any **C compiler** (like Code::Blocks, Dev-C++, or GCC).  
2. Compile the program.  
3. Run the executable.  
4. Follow on-screen prompts to perform calculations.
## For HTML/CSS Interface:
1. Open `index.html` in your **web browser**.  
2. Use the calculator buttons
